HTML 코드 작성 규칙

HTML4 Transitional

  • 표준 DTD 문서 HTML4 관용모드(Transitional) 기본 골격 사용.
  • meta 태그 http-equiv="X-UA-Compatible" content="IE=edge" 사용으로 최신모드로 지정된 DOCTYPE에 상관없이 IE8 이상 버전에서 항상 최신 표준 모드로 렌더링되도록 노출방식 지정.
							<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
							<html lang="en">
							<head>
								<meta http-equiv="Content-Type" content="text/html;charset=UTF-8">
								<meta http-equiv="X-UA-Compatible" content="IE=edge">
								<title>Document</title>
							</head>
							<body>
								...
							</body>
							</html>
